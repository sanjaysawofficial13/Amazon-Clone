 # 🛒 Amazon Clone

A responsive Amazon Clone built using modern web technologies.
This project replicates the core UI and basic functionality of Amazon’s homepage, including product listings, navigation bar, and cart interface.

📌 Features

🏠 Amazon-style Homepage UI

🔍 Search Bar

🛍 Product Listings

🛒 Add to Cart UI

📱 Fully Responsive Design

🎨 Clean Layout & Styling

🛠 Tech Stack

Frontend: HTML5, CSS3,Bootstrap JavaScript
Deployment: GitHub Pages / Netlify

⚙️ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/your-username/amazon-clone.git

2️⃣ Navigate to the folder

cd amazon-clone

3️⃣ Open index.html in your browser
OR use Live Server in VS Code.
